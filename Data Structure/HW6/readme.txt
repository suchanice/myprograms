*  Description of the program
*******************************************************

The program "DictEnty" reads words from a data file. When complete print out each word with spaces in between in a file called revsorted.txt. Start at the back of the list so you end up with a reverse sorted list 


*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program.  This is the driver program that calls sub-functions
   to read data from an input file, use the data to create two matrices,
   and compute and display their sum and product.

Name:  DictEntry.h
   Contains the definition for the class DictEntry.  

Name: DictEntry.cpp
   Defines and implements the matrix class for implementing a DictEntry.
